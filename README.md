###Spring 2014 IDC
------
##How to git
---
#Get the repository
```
git clone https://github.com/saicheems/IDC.git
```
#Get the up to date repository
```
git pull origin master
```
#Add a new file/folder to be committed
```
git add FILENAME
```
#Commit
```
git commit -m 'ADD YOUR MESSAGE'
```
#Push to github
```
git push origin master
```
